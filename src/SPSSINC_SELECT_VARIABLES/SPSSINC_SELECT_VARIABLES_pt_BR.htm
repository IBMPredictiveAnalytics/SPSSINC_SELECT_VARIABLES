Definir uma macro que lista variáveis selecionadas

Este comando fornece uma maneira de definir uma macro do SPSS cujo valor é uma lista de variáveis que satisfazem critérios como nível de medição, tipo, atributos de variáveis e padrões nos nomes.

Usando essa abordagem, é possível criar tarefas de sintaxe generalizadas que não dependem de conhecer os nomes de variáveis exatos em um arquivo de dados específico. Por exemplo, você poderia criar uma tarefa que exibe estatísticas descritivas apropriadas para as variáveis em um conjunto de dados com base em seu nível de medição.

Nome da Macro. Insira um nome para a macro a ser gerada. O convencional é iniciar o nome com "!" para evitar conflito com qualquer nome de variável.

Variáveis e Limitar seleção a. Mova variáveis para o controle Limitar seleção a para restringir o resultado a um subconjunto de somente aqueles nomes. Se nenhum outro critério for especificado, a macro deve definir exatamente esses nomes.

•Se o controle Limite for deixado em branco, os outros critérios serão aplicados a todas as variáveis.

Tipo de variável. Limite a seleção a variáveis numéricas, variáveis de sequência de caracteres ou a ambos os tipos (sem limite).

Nível de variável. Selecione os critérios do nível de medição de variável a aplicar.

Função da variável. No Statistics versão 18 ou posterior, variáveis podem ter papéis designados. Selecione o critério de função a aplicar. Na versão 17, este critério será ignorado.

•A função é especialmente útil se este diálogo for usado no Modelador.

Expressão regular de nome de variável. É possível inserir uma expressão regular para filtrar as variáveis com base nos nomes. As expressões regulares descrevem um padrão de caracteres. O padrão é sem distinção entre maiúsculas e minúsculas e inicia no início do nome. Exemplos:

age - qualquer variável iniciada por age

.*age - qualquer variável cujo nome contenha age

.*\d$ - qualquer variável cujo nome termine em um dígito

Atributos.

A caixa de subdiálogo Atributos permite que você especifique os critérios com base nos atributos customizados. Atributos customizados são propriedades de variáveis que podem ser criadas pelo comando VARIABLE ATTRIBUTE, pela caixa de diálogo Definir propriedades da variável ou no Editor de dados. Eles são salvos com o conjunto de dados.

As variáveis devem ter todos estes atributos. Variáveis selecionadas devem ter todos os atributos denominados aqui. Insira um ou mais nomes de atributos separados por um espaço em branco. O valor do atributo, se houver, não importa.

As variáveis devem ter pelo menos um destes atributos. As variáveis irão satisfazer este critério se tiverem pelo menos um dos atributos listados. Insira um ou mais nomes de atributos separados por um espaço em branco. O valor do atributo, se houver, não importa. Os nomes não fazem distinção entre maiúsculas e minúsculas.

Valores de atributos. Insira nomes de atributos e listas de valores para restringir as variáveis selecionadas àquelas com estes valores. É possível inserir até três conjuntos de nomes e valores.

•Os valores devem ser cada um colocado entre aspas simples ou duplas e, se houver mais de um valor, separe-os por um espaço em branco.

•Os valores fazem distinção entre maiúsculas e minúsculas: se um atributo variável denominado units tiver um valor igual a EUROS, você deve inserir "EUROS" no campo de valor para units para selecionar a variável.

•Se a variável não tiver um atributo especificado, o teste será ignorado. Liste o atributo no controle As variáveis devem ter todos estes atributos para excluir as variáveis que não têm o atributo.

•Se o atributo for um atributo de matriz, em que pode haver muitos valores para um único atributo, a variável satisfaz os critérios se qualquer valor na matriz corresponder a um valor na lista Valores de atributos.

Opções

Ordem das variáveis selecionadas na macro. É possível especificar se a lista de variáveis definida pela macro é alfabética ou de acordo com a ordem das variáveis no arquivo. Se Conforme listado for selecionado, quaisquer variáveis na lista Limitar seleção a que passem nas outras condições serão exibidas na ordem listada na definição de macro.

Se nenhuma variável estiver selecionada. Se nenhuma variável satisfizer os critérios de seleção, é possível levantar um erro e não definir a macro ou é possível continuar e gerar uma macro vazia. A macro vazia poderia então ser testada posteriormente usando o código de macro para evitar a geração de erros devido a ser vazia.

Imprimir definição de macro. Escolha se o conteúdo da macro é listado em uma tabela dinâmica ou não.

Caractere separador. Escolha se deseja separar os nomes das variáveis na definição de macro com um espaço em branco ou um sinal de mais. O sinal de mais é útil para expressões TABLE de Tabelas customizadas.

Recursos adicionais

Execute o comando
SPSSINC SELECT VARIABLES /HELP.
para exibir a descrição toda da sintaxe.

Para os programadores de Python, um dicionário das macros definidas por este comando pode ser acessado como SPSSINC_SELECT_VARIABLES.macrodict.

Uma lista de papéis pode ser especificada.

TO e ALL podem ser usados em sintaxe juntamente com as variáveis específicas.

Requisitos

Este comando requer o Python Plug-in Programmability. O Plug-in pode ser localizado no CD do IBM SPSS Statistics, ou você pode fazer download dele no SPSS Developer Central (www.spss.com/devcentral).
