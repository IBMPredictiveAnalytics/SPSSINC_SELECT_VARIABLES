선택한 변수를 나열하는 매크로 정의

이 명령문은 측정 수준, 유형, 변수 속성 및 이름 패턴 등의 기준을 충족하는 변수 목록의 값이 포함된 SPSS 매크로를 정의하는 방법을 제공합니다.

이 방법을 사용하여 특정 데이터 파일의 정확한 변수 이름을 알지 않아도 일반화된 명령문 작업을 작성할 수 있습니다. 예를 들어, 측정 수준에 따라 데이터 세트의 변수에 대한 적절한 설명 통계를 표시하는 작업을 작성할 수 있습니다.

매크로 이름. 생성할 매크로의 이름을 입력하십시오. 일반적으로 각 매크로 이름은 변수 이름과의 충돌을 피하기 위해 이름이 "!"로 시작됩니다.

다음에 대한 변수 및 선택항목 제한. 결과를 해당 이름의 부분집합으로 제한하려면 변수를 다음에 대한 선택항목 제한 제어로 이동하십시오. 다른 기준을 지정하지 않은 경우 매크로가 해당 이름을 정확하게 정의합니다.

•한계 제어가 비어 있으면 다른 기준이 모든 변수에 적용됩니다.

변수 유형. 선택을 숫자변수, 문자변수 또는 둘 모두(제한 없음)로 제한하십시오.

변수 수준. 적용할 변수 측정 수준 기준을 선택하십시오.

변수 역할. Statistics 버전 18 이상에서는 변수에 역할이 지정될 수 있습니다. 적용할 역할 기준을 선택하십시오. 버전 17에서는 이 기준이 무시됩니다.

•특히, 이 대화 상자가 Modeler에 사용될 경우 이 역할이 유용합니다.

변수 이름 규칙 표현식. 정규식을 입력하여 이름을 기준으로 변수를 필터링할 수 있습니다. 정규식은 문자 패턴을 설명합니다. 패턴은 대소문자를 구분하지 않으며 이름 시작 부분에서 시작합니다. 예를 들면 다음과 같습니다.

age - age로 시작하는 모든 변수

.*age - 이름에 age가 포함된 모든 변수

.*\d$ - 이름이 숫자로 끝나는 모든 변수

속성.

속성 하위 대화 상자를 사용하여 사용자 정의 속성 기준을 지정할 수 있습니다. 사용자 정의 속성은 VARIABLE ATTRIBUTE 명령문, 변수특성정의 대화 상자 또는 데이터 편집기로 작성할 수 있는 변수의 특성입니다. 이러한 특성은 데이터 세트로 저장됩니다.

변수에는 이러한 모든 속성이 포함되어야 합니다. 선택한 변수에는 여기에서 이름 지정한 모든 속성이 포함되어야 합니다. 하나 이상의 속성 이름을 공백으로 구분하여 입력하십시오. 속성 값(있는 경우)은 문제가 되지 않습니다.

변수에는 이러한 모든 속성이 하나 이상 포함되어야 합니다. 하나 이상의 속성이 나열된 변수는 이 기준을 충족합니다. 하나 이상의 속성 이름을 공백으로 구분하여 입력하십시오. 속성 값(있는 경우)은 문제가 되지 않습니다. 이름은 대소문자를 구분하지 않습니다.

속성 값. 선택한 변수를 이러한 값이 있는 변수로 제한하려면 속성 이름 및 값 목록을 입력하십시오. 이름 및 값 세트를 최대 3개까지 입력할 수 있습니다.

•값은 각각 작은따옴표 또는 큰따옴표로 묶어야 합니다. 값이 여러 개인 경우 공백으로 구분하십시오.

•값은 대소문자를 구분합니다. units라는 변수 속성에 EUROS 값이 있는 경우 변수를 선택하려면 단위 값 필드에 "EUROS"를 입력해야 합니다.

•변수에 지정된 속성이 없으면 검정이 무시됩니다. 속성이 없는 변수를 제외하려면 변수에는 이러한 모든 속성이 포함되어야 합니다. 제어에 속성을 나열하십시오.

•속성이 단일 속성에 대한 값이 여러 개 있을 수 있는 배열 속성인 경우 배열의 값이 속성 값 목록의 값과 일치하면 변수가 기준을 충족하는 것입니다.

옵션

매크로에서 선택된 변수의 순서. 매크로를 통해 정의된 변수 목록을 알파벳순 또는 파일의 변수 순서에 따라 나열되도록 지정할 수 있습니다. 목록별을 선택할 경우 다른 조건을 전달하는 목록에 대한 선택항목 제한의 모든 변수가 매크로 정의에 나열된 순서대로 표시됩니다.

변수가 선택되지 않은 경우. 변수가 선택기준을 충족하지 않는 경우 오류를 생성하고 매크로를 정의하지 않을 수 있습니다. 또는 빈 매크로를 처리하여 생성할 수도 있습니다. 그런 다음 빈 매크로로 인해 오류가 생성되는 것을 방지하려면 나중에 매크로 코드를 사용하여 빈 매크로를 검정할 수 있습니다.

매크로 정의 출력. 피벗표에 매크로의 컨텐츠를 나열할지 여부를 선택하십시오.

구분자. 공백 또는 더하기 기호를 사용하여 매크로 정의의 변수 이름을 구분할지 여부를 선택하십시오. 사용자 정의 표의 TABLE 표현식의 경우에는 더하기 기호가 유용합니다.

추가 기능

다음 명령문을 실행하십시오.
SPSSINC SELECT VARIABLES /HELP.
전체 명령문 설명을 표시합니다.

Python 프로그래머의 경우 명령문으로 정의된 매크로 사전에 SPSSINC_SELECT_VARIABLES.macrodict로 액세스할 수 있습니다.

역할 목록을 지정할 수 있습니다.

TO 및 ALL을 특정 변수와 함께 명령문에서 사용할 수 있습니다.

요구 사항
