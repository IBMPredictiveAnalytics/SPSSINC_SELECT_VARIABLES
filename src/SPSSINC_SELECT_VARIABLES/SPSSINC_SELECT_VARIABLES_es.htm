Definir una macro que ofrezca una lista de las variables seleccionadas

Este comando ofrece una forma de definir una macro de SPSS cuyo valor es una lista de variables que cumplen criterios como nivel de medición, tipo, atributos de variables y patrones en los nombres.

Mediante este método, puede crear trabajos de sintaxis generalizados que no dependan de conocer los nombres exactos de variables en un archivo de datos determinado. Por ejemplo, puede crear un trabajo que muestre estadísticos descriptivos adecuados para las variables de un conjunto de datos en función de su nivel de medición.

Nombre de macro. Especifique un nombre para la macro que se va a generar. Es una convención empezar el nombre con "!" para evitar conflictos con otros nombres de variables.

Variables y Limitar la selección a. Mueva las variables al control Limitar la selección a para restringir el resultado a un subconjunto de sólo esos nombres. Si no se especifica ningún otro criterio, la macro definiría esos nombres con exactitud.

•Si el control de límite se deja vacío, el otro criterio se aplicaría a todas las variables.

Tipo de variable. Limita la selección a variables numéricas, variables de cadena o a los dos tipos (sin límite).

Nivel de variable. Seleccione el criterio de nivel de medición de variable que se va a aplicar.

Función de variable. En Statistics versión 18 o posterior, las variables pueden tener funciones asignadas. Seleccione el criterio de función que vaya a aplicar. En la versión 17, este criterio se ignora.

•La función resulta particularmente útil si este cuadro de diálogo se va a utilizar en Modeler.

Expresión regular de nombre de variable. Puede especificar expresión regular para filtrar las variables en función de los nombres. Las expresiones regulares describen un patrón de caracteres. El patrón no es sensible a las mayúsculas y minúsculas y se inicia al principio del nombre. Ejemplos:

age - cualquier variable que empiece por age

.*age - cualquier variable cuyo nombre contenga age

.*\d$ - cualquier variable cuyo nombre termine con un dígito

Atributos

El subcuadro de diálogo Atributos permite especificar criterios en función de los atributos personalizados. Los atributos personalizados son propiedades de variables que se pueden crear mediante el comando VARIABLE ATTRIBUTE, mediante el cuadro de diálogo Definir propiedades de variables o en el Editor de datos. Se guardan con el conjunto de datos.

Las variables deben tener todos estos atributos. Las variables seleccionadas deben tener todos los atributos nombrados aquí. Escriba uno o más nombres de atributos separados por un espacio en blanco. El valor del atributo, de haber alguno, no importa.

Las variables deben tener al menos uno de estos atributos. Las variables cumplirán este criterio si tienen al menos uno de los atributos enumerados. Escriba uno o más nombres de atributos separados por un espacio en blanco. El valor del atributo, de haber alguno, no importa. Los nombres no son sensibles a mayúsculas y minúsculas.

Valores del atributo. Escriba nombres de atributo y listas de valores para restringir las variables seleccionadas a aquellas que tengan esos valores. Puede especificar hasta tres conjuntos de nombres y valores.

•Cada valor debe estar entre comillas dobles o sencillas y, si hay más de un valor, debe separarlos por un espacio en blanco.

•Los valores son sensibles a las mayúsculas y minúsculas: si un atributo de variable llamado unidades tiene un valor de EUROS, debe especificar "EUROS" en el campo de valor de las unidades para poder seleccionar la variable.

•Si la variable no tiene un atributo especificado, se ignora la prueba. Incluya el atributo en el control Las variables deben tener todos estos atributos para poder excluir las variables que no tengan el atributo.

•Si el atributo es un atributo matriz, en el que puede haber varios valores para un único atributo, la variable cumple el criterio si cualquier valor de la matriz coincide con un valor de la lista Valores del atributo.

Opciones

Orden de las variables seleccionadas en la macro. Puede especificar que la lista de variables definida por la macro tenga un orden alfabético o bien siga el orden de las variables del archivo. Si selecciona Según lista, las variables de la lista Limitar la selección a que pasan las otras condiciones aparecerán en el orden de la lista de la definición de la macro.

Si no se seleccionan variables. Si ninguna variable cumple el criterio de selección, puede generar un error y no definir la macro, o puede continuar y generar una macro vacía. La macro vacía puede probarse posteriormente mediante un código de macro para evitar que se generen errores por estar vacía.

Imprimir definición de macro. Elija si el contenido de la macro se va a listar en una tabla dinámica.

Carácter separador. Elija si va a separar los nombres de variables en la definición de la macro por un espacio en blanco o un signo más. El signo más es útil en las expresiones TABLE de las Tablas personalizadas.

Características adicionales

Ejecute el comando
SPSSINC SELECT VARIABLES /HELP.
para mostrar toda la descripción de la sintaxis.

Los programadores de Python pueden acceder a un diccionario de macros definidas mediante este comando como SPSSINC_SELECT_VARIABLES.macrodict.

Se puede especificar una lista de funciones.

Se puede utilizar TO y ALL en la sintaxis junto con variables específicas.

Requisitos

Este comando requiere el complemento de programabilidad Python. Puede encontrar el complemento Python en el CD de IBM SPSS Statistics o puede descargarlo desde SPSS Developer Central (www.spss.com/devcentral).
