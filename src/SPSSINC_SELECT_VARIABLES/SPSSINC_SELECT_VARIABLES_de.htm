Makro zur Auflistung ausgewählter Variablen definieren

Dieser Befehl ermöglicht das Definieren eines SPSS-Makros, dessen Wert aus einer Liste von Variablen besteht, die bestimmte Kriterien erfüllen (wie Messniveau, Typ, Variablenattribute und Muster in den Namen).

Auf diese Weise können Sie verallgemeinerte Syntaxjobs erstellen, bei denen es nicht erforderlich ist, dass die genauen Variablennamen in einer bestimmten Datendatei bekannt sind. Beispielsweise könnten Sie einen Job erstellen, der entsprechende deskriptive Statistiken für die Variablen in einem Dataset auf der Basis ihrer Messniveaus anzeigt.

Makroname. Geben Sie einen Namen für das zu generierende Makro ein. Der Konvention folgend sollte ein Ausrufezeichen (!) als erstes Zeichen des Namens verwendet werden, um Konflikte mit Variablennamen zu vermeiden.

'Variablen' und 'Auswahl begrenzen auf'. Verschieben Sie Variablen in das Steuerelement Auswahl begrenzen auf, um das Ergebnis auf ein nur aus diesen Namen bestehendes Subset zu begrenzen. Unter der Voraussetzung, dass keine anderen Kriterien angegeben sind, würde das Makro genau diese Namen definieren.

•Wenn das Begrenzungssteuerelement leer gelassen wird, werden die anderen Kriterien auf alle Variablen angewendet.

Variablentyp definieren. Begrenzen Sie die Auswahl auf numerische Variablen, Zeichenfolgevariablen oder auf beide Typen (keine Begrenzung).

Variablenniveau. Wählen Sie die für das Variablenmessniveau anzuwendenden Kriterien aus.

Variablenrolle. Ab Statistics Version 18 können Variablen Rollen zugewiesen werden. Wählen Sie das anzuwendende Rollenkriterium aus. In Version 17 wird dieses Kriterium ignoriert.

•Die Rolle ist besonders hilfreich, wenn dieses Dialogfeld in Modeler verwendet werden soll.

Regulärer Ausdruck für Variablennamen. Sie können einen regulären Ausdruck eingeben, um die Variablen auf der Basis der Namen zu filtern. Reguläre Ausdrücke beschreiben ein Zeichenmuster. Das Muster, bei dem die Groß-/Kleinschreibung nicht beachtet werden muss, beginnt mit dem ersten Zeichen des Namens. Beispiele:

alter - alle Variablen, die mit der Zeichenfolge 'alter' beginnen

.*alter - alle Variablen, in deren Namen die Zeichenfolge 'alter' vorkommt

.*\d$ - alle Variablen mit einem Namen, dessen letztes Zeichen eine Ziffer ist

Attribute.

Im Unterdialogfeld 'Attribute' können auf der Basis von benutzerdefinierten Attributen Kriterien angegeben werden. Benutzerdefinierte Attribute sind Eigenschaften von Variablen, die mit dem Befehl VARIABLE ATTRIBUTE, Dialogfeld 'Variableneigenschaften definieren' oder im Dateneditor erstellt werden können. Sie werden mit dem Dataset gespeichert.

Variablen müssen alle angegebenen Attribute aufweisen. Die ausgewählten Variablen müssen alle der hier genannten Attribute aufweisen. Geben Sie einen oder mehrere durch ein Leerzeichen voneinander getrennte Attributnamen ein. Der Wert des Attributs, sofern vorhanden, ist irrelevant.

Variablen müssen mindestens eines der angegebenen Attribute aufweisen. Variablen erfüllen dieses Kriterium, wenn sie mindestens eines der aufgelisteten Attribute aufweisen. Geben Sie einen oder mehrere durch ein Leerzeichen voneinander getrennte Attributnamen ein. Der Wert des Attributs, sofern vorhanden, ist irrelevant. Bei den Namen muss die Groß-/Kleinschreibung nicht beachtet werden.

Attributwerte. Geben Sie Attributnamen und Wertelisten ein, um die ausgewählten Variablen auf diejenigen zu beschränken, die diese Werte aufweisen. Eingegeben werden können bis zu drei Sets von Namen und Werten.

•Jeder Wert muss in einfache oder doppelte Anführungszeichen eingeschlossen und, wenn mehrere Werte vorhanden sind, müssen diese durch ein Leerzeichen voneinander getrennt werden.

•Bei den Werten muss die Groß-/Kleinschreibung beachtet werden: wenn ein Variablenattribut mit dem Namen einheiten den Wert EUROS hat, muss "EUROS" in das Wertfeld für Einheiten eingegeben werden, um die Variable auszuwählen.

•Weist die Variable kein angegebenes Attribut auf, wird der Test ignoriert. Listen Sie das Attribut im Steuerelement Variablen müssen alle angegebenen Attribute aufweisen auf, um Variablen auszuschließen, die das Attribut nicht aufweisen.

•Handelt es sich bei dem Attribut um ein Array-Attribut, bei dem ein einzelnes Attribut über mehrere Werte verfügen kann, erfüllt die Variable die Kriterien, wenn eines der Werte im Array mit einem der Werte in der Liste Attributwerte übereinstimmt.

Optionen

Reihenfolge der ausgewählten Variablen im Makro. Sie können angeben, dass die vom Makro definierte Variablenliste in alphabetischer Reihenfolge oder in der Reihenfolge der Variablen in der Datei sortiert sein soll. Bei Auswahl von 'Wie angegeben' werden alle in der Liste 'Auswahl begrenzen auf' vorhandenen Variablen, die die anderen Bedingungen erfüllen, in der Reihenfolge dargestellt, die in der Makrodefinition angegeben ist.

Wenn keine Variablen ausgewählt sind. Wenn keine Variablen die Auswahlkriterien erfüllen, können Sie entweder einen Fehler ausgeben und das Makro nicht definieren oder Sie können mit dem Vorgang fortfahren und ein leeres Makro generieren. Das leere Makro könnte dann mithilfe des Makrocodes zu einem späteren Zeitpunkt getestet werden, um zu vermeiden, dass Fehler generiert werden, weil das Makro leer ist.

Makrodefinition ausgeben. Wählen Sie aus, ob der Inhalt des Makros in einer Pivot-Tabelle aufgelistet werden soll.

Trennzeichen. Wählen Sie aus, ob Variablennamen in der Makrodefinition durch ein Leerzeichen oder ein Pluszeichen voneinander getrennt werden sollen. Das Pluszeichen ist bei TABLE-Ausdrücken bei benutzerdefinierten Tabellen nützlich.

Weitere Funktionen

Führen Sie den folgenden Befehl aus:
SPSSINC SELECT VARIABLES /HELP.
Sie zeigen damit die vollständige Syntaxbeschreibung an.

Hinweis für Python-Programmierer: Über SPSSINC_SELECT_VARIABLES.macrodict besteht Zugriff auf ein Wörterbuch mit den über diesen Befehl definierten Makros.

Es besteht die Möglichkeit, eine Liste von Rollen anzugeben.

TO und ALL können in der Syntax zusammen mit bestimmten Variablen verwendet werden.

Anforderungen

Dieser Befehl erfordert das Python-Programmierungs-Plug-in. Sie finden das Python-Plug-in auf der IBM SPSS Statistics-CD oder Sie können es über SPSS Developer Central (www.spss.com/devcentral) herunterladen.
